# 🌐 Portfolio Website

## 📌 Description
This is a personal **portfolio website** built using **React.js, HTML, and CSS**. The website is designed to showcase my projects, technical skills, and development experience in a clean and responsive layout.

The goal of this project is to create a professional online presence where visitors, recruiters, and collaborators can explore my work and learn more about my development capabilities.

The application uses **React.js** to build reusable UI components and manage the frontend structure, while **HTML and CSS** are used to create a modern, responsive, and user-friendly interface.

---

## 🚀 Features
- Responsive portfolio layout
- Projects showcase section
- Skills section
- About me section
- Contact section
- Clean and modern UI design

---

## 🛠 Tech Stack

- CSS3
- JavaScript
